📡 Endpoints

Todos los endpoints son públicos, no requieren autenticación.

GET
/health
Verifica que la API esté en línea.
 GET
/api/states
Retorna todos los estados de Venezuela con imágenes.
 GET
/api/states/:id
Retorna un estado específico con sus ciudades, municipios e imágenes.
Ej: /api/states/1
 GET
/api/cities
Retorna las primeras 50 ciudades con su estado relacionado.
 GET
/api/cities/:id
Retorna una ciudad específica con su estado e imágenes.
Ej: /api/cities/1
 GET
/api/search?q=texto
Busca estados y ciudades por nombre.
Ej: /api/search?q=Caracas

💻 Ejemplos de uso

Cómo consumir la API desde diferentes lenguajes.

# Listar todos los estados
curl https://venezuela-api-two.vercel.app/api/states

# Obtener un estado por ID
curl https://venezuela-api-two.vercel.app/api/states/1

# Buscar por nombre
curl "https://venezuela-api-two.vercel.app/api/search?q=Caracas"
// Listar todos los estados
const response = await fetch('https://venezuela-api-two.vercel.app/api/states');
const states = await response.json();
console.log(states);

// Buscar por nombre
const res = await fetch('https://venezuela-api-two.vercel.app/api/search?q=Caracas');
const data = await res.json();
console.log(data);
import axios from 'axios';

// Listar todos los estados
const { data } = await axios.get('https://venezuela-api-two.vercel.app/api/states');
console.log(data);

// Obtener estado por ID
const { data: state } = await axios.get('https://venezuela-api-two.vercel.app/api/states/1');
console.log(state);
import requests

# Listar todos los estados
response = requests.get('https://venezuela-api-two.vercel.app/api/states')
states = response.json()
print(states)

# Buscar por nombre
response = requests.get(
    'https://venezuela-api-two.vercel.app/api/search',
    params={'q': 'Caracas'}
)
print(response.json())
